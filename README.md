## How to run the program
### Step1: Clone my project from the repo
```
git clone https://github.com/Nightsky-na/project-phase1.git
```

### Step2: Install node_modules
```
npm install
```

### Step3: Build the tailwindcss
```
npm run build
```

### Step4: Start the node server
```
npm start
```
